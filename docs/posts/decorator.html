  • 重要準則:

    @f
def g():
  ...

    等價於

    g = f(g)

使用方式

定義如下的 add_to_class.

def add_to_class(Class):
  def wrapper(obj):
    setattr(Class, obj.__name__, obj)
  return wrapper

假如我們有一個 class A, 且有個 method double.

class A:
  def double(self,t):
    return 2*t

令 a 為 A 的 instance.

a = A()

則我們可以呼叫 a.double(3).

a.double(3)
6

現在假如我們想修改 A 的 method double, 可用如下.

@add_to_class(A)
def double(self,t):
  return 7*t

則可以發現: 不需重新令 a 是 A 的 instance, a 的 double 都已經變更.

a.double(3)
21
# 另一個 A 的 instance
aa = A()
aa.double(3)
21

解析 code

回顧 add_to_class 為

def add_to_class(Class):
  def wrapper(obj):
    setattr(Class, obj.__name__, obj)
  return wrapper

且

@add_to_class(A)
def double(self,t):
  return 7*t

等價於 double = add_to_class(A)(double).

Important

這裡會令 double 為 add_to_class(A)(double). 所以要注意不會覆蓋到舊有的變數. 可以單獨執行 double 會發現不報錯 (不會 return 任何東西).

double, type(double)
(None, NoneType)

所以有執行了 add_to_class(A)(double)1 這指令.

  • 1 這裡 add_to_class(A) 不 是 class, 而是 wrapper.

    • 所以有執行 setattr(Class, obj.__name__, obj), 其中 Class = A, obj = double.

    所以相當於執行了 setattr(A, double, 7*t)23.

  • 2 這裡的 obj.__name__ 相當於 obj 的名字. 在這就是 double.

  • 3 這裡 setattr(Class, obj.__name__, obj) 的第三分量 obj 相當於 double return 的東西, 所以就是 7*t.
